# Stock Prediction Using LSTM

This project uses Python and an LSTM neural network to predict Apple (AAPL) stock closing prices.

## What This Notebook Does

1. Downloads stock data using Yahoo Finance.
2. Performs simple analysis and plots for AAPL, GOOG, MSFT, and AMZN.
3. Trains an LSTM model on AAPL historical close prices.
4. Predicts prices and compares predicted vs actual values.

## File

- `stock_pred.ipynb`: Main notebook with data collection, visualization, model training, and prediction.

## Libraries Used

- pandas
- numpy
- matplotlib
- seaborn
- yfinance
- pandas-datareader
- scikit-learn
- keras / tensorflow

## How to Run

1. Install required libraries.
2. Open `stock_pred.ipynb` in Jupyter or VS Code.
3. Run cells from top to bottom.
